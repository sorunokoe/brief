# Archived Examples

These examples demonstrate Brief's full feature set but are not maintained as canonical end-to-end runnable examples.

The canonical examples are in the parent `examples/` directory:

| File | What it shows |
|------|---------------|
| `01-hello.brief` | Minimum valid task |
| `02-review-pr.brief` | GitHub + FileSystem workflow |
| `15-checkout-flow.brief` | `needs {}` + `forbids {}` scope enforcement |
| `28-ai-pipeline.brief` | LLM + embeddings + RAG pipeline |

The archived examples here compile and type-check (`brief check`), but they reference skill MCP servers
that may not be available in your environment.
